# Elevate Recruitment Platform 🚀

Elevate is a modern, full-stack recruitment platform designed to streamline the hiring process for both candidates and recruiters. Built with React, Vite, and Tailwind CSS, it offers a seamless experience for managing job offers, applications, and interviews.

## ✨ Features

### 👤 For Candidates
- **Job Board**: Browse and search through available job opportunities.
- **Application Tracking**: Monitor the status of your applications in real-time.
- **Interview Management**: View and keep track of scheduled interviews.
- **Professional Profile**: Manage your personal information and resume.

### 💼 For Recruiters
- **Dashboard**: Get a high-level overview of recruitment activities and statistics.
- **Job Management**: Create, edit, and manage job offers easily.
- **Application Manager**: Review candidate applications, update statuses, and move candidates through the pipeline.
- **Interview Scheduling**: Organize and manage interviews with top talent.

### 🛡️ Admin
- Full access to oversee the platform's operations and users.

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite 8, TypeScript
- **Styling**: Tailwind CSS 4
- **Icons**: Lucide React
- **Routing**: React Router 7
- **API Communication**: Axios
- **State/Auth**: Context API with Private & Role-based Routing

## 🚀 Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/TORBIomar/Elevate.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## 📂 Project Structure

- `src/components`: Reusable UI components and route guards.
- `src/contexts`: Authentication and state management.
- `src/layouts`: Page layouts (Auth and Main).
- `src/pages`: Feature-specific pages (Auth, Jobs, Recruiter, Candidate).
- `src/services`: API services for backend communication.
- `src/types`: TypeScript interfaces and types.
